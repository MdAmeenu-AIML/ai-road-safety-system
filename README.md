# 🚗 AI Road Safety & Accident Alert System

An AI-powered road safety dashboard built using Python, Machine Learning, Streamlit, and interactive maps.

This project analyzes accident data, detects high-risk areas, visualizes dangerous locations on maps, and predicts accident risk levels using Machine Learning.

## 📊 Dataset

The dataset contains:

- Accident counts

Example:

|---|---|---|
| Delhi | 300 | High |
| Mumbai | 200 | High |
| Pune | 90 | Low |

## 🤖 Machine Learning Workflow

1. Load accident dataset
2. Analyze accident statistics
3. Visualize accident-prone areas
4. Train Decision Tree Classifier
5. Predict accident risk levels
6. Display predictions on dashboard

## ▶️ Run Locally

### Clone Repository

```bash
git clone https://github.com/MdAmeenu-AIML/ai-road-safety-system.git
```

### Open Project Folder

```bash
cd ai-road-safety-system
```

### Install Requirements

```bash
pip install -r requirements.txt
```

### Run Streamlit App

```bash
streamlit run app.py
```
